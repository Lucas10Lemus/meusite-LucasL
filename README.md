## Requisitos
- Node.js >= 16
- npm (ou yarn)

## Como usar

1. **Clonar o repositório**
```powershell
git clone https://github.com/seu-usuario/tron-portfolio.git
cd tron-portfolio
```

2. **Instalar dependências**
```powershell
npm install
```

3. **Rodar em modo de desenvolvimento**
```powershell
npm run dev
```
Após executar o comando, abra seu navegador em `http://localhost:3000` (ou na porta que aparecer no terminal).

4. **Build para produção**
```powershell
npm run build
```
Os arquivos otimizados para produção serão gerados na pasta `./dist` (ou `./build`).

## Testes
Para rodar os testes automatizados, execute:
```powershell
npm test
```

## Estrutura de Pastas
- `src/`: Código-fonte da aplicação.
- `public/`: Arquivos estáticos (imagens, fontes, etc.).
- `dist/`: Arquivos gerados para produção após o build.
- `tests/`: Testes automatizados.
